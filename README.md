# Rat

The [cat](https://www.gnu.org/software/coreutils/cat) clone on meth, inspired by [bat](https://github.com/sharkdp/bat)

## Installation

```shell
cmake .
make
sudo make install
```

## Usage

Use `rat --help` to see all available options
